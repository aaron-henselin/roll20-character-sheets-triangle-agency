# Triangle Agency Roll20 Character Sheet

An unofficial Roll20 character sheet for **Triangle Agency**.

Triangle Agency is designed by Caleb Zane Huett and Sean Ireland and published
by Haunted Table Games. This Roll20 sheet is an unofficial fan/community
conversion by Aaron Henselin. It is not affiliated with, endorsed by, or
sponsored by Haunted Table Games or the Triangle Agency creators.

## Included Files

- `triangle-agency.html`: Roll20 sheet HTML and Sheet Workers.
- `triangle-agency.css`: Roll20 sheet styling.
- `sheet.json`: Roll20 sheet metadata.

## Sheet Features

- Agent identity, ARC selection, mission metrics, Quality Assurances, and
  Work/Life Balance tracking.
- Fixed Relationship and Requisition/Work/Life Benefit slots matching the
  character sheet structure.
- Structured Anomaly abilities with rules text, Be Known answer tracks, and
  persisted checkboxes.
- Ability invocation rolls using Triangle Agency's six-d4 success check,
  including Burnout handling and Roll20 roll template output.
- Intake and Post-Mission tabs for campaign bookkeeping.

## Notes

This sheet is intended to help play Triangle Agency in Roll20. It does not
replace the official game text. Players and GMs should own or otherwise have
access to the official Triangle Agency rules.

Some sheet visuals are recreated with Roll20-compatible HTML and CSS rather
than the original print assets, so the sheet is usable without external image
requirements.
